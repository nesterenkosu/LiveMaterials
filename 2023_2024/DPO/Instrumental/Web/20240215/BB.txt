Отладка PHP
~~~~~~~~~~~
- Эмулятор веб-сервера (Denwer, OpenServer, XAMPP ...)
- Реальное ПО веб-сервера, установленное на локальном компьютере
- Отладка кода непосредственно на веб-хостинге

Базы данных на веб-сайтах
~~~~~~~~~~~~~~~~~~~~~~~~~
Любой современный веб-сайт почти всегда использует базы данных
Популярные СУБД
PHP
- MySQL
- MariaDB
- PostgreSQL
ASP.NET
- MS SQL Server

Связка LAMP - Linux Apache MySQL PHP - "классическая" связка для веб-сайтов на PHP

Базовые SQL-конструкции

INSERT - добавление новых данных
SELECT - извлечение существующих данных
UPDATE - обновление существующих данных
DELETE - удаление данных

SQL-инъекции и защита от них

Способы защиты:
Обработка текстового ввода функцией mysqli_real_escape_string

Явное приведение числовых строк к числовым типам данных (int, double)

Включение magic_quotes в настройках сервера

Использование "продвинутых" движков доступа к данным, например PDO (Php Data Objects).
