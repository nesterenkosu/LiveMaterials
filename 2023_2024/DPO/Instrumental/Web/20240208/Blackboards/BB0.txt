Web-программирование:
- Серверное (серверное ПО выполняется на сервере, в облаке)
PHP, Python, ASP.NET, NodeJS, Ruby ...
- Клиентское (клиентское ПО выполняется у клиента на компьютере в браузере). Клиентские языки программирования: HTML, CSS, JavaScript

Любой современный веб-сайт на серверной стороне использует базы данных (СУБД) - MySQL, MariaDB, MS SQL ...

Фреймворки - надстройки над клиентскими и серверными языками, облегчающие работу с ними
JavaScript - jQuery, Vue, ReactJS ...
CSS - Bootstrap
PHP - Laravel, Symphony, Code Igniter ...

Роли в команде разработчиков веб-сайта:
- Дизайнер - создаёт макет внешнего вида сайта в графическом редакторе (например, Photoshop, Figma)
- Верстальщик - тот, кто переводит графический макет, созданный дизайнером, в программный код на языках HTML, CSS, JS
- Программист - пишет программный код логики приложения. Создаёт базу данных и определяет логику работы с ней
- SEO-оптимизатор - занимается продвижением сайта в поисковых системах.

Веб-хостинг - удалённый компьютер, на котором находятся файлы сайта, и с которого обеспечивается доступ к сайту со всего мира.



Клиентские веб-языки
~~~~~~~~~~~~~~~~~~~~
HTML

ПО для написания HTML
- Notepad++, Visual Studio, Visual Studio Code, Sublime, Atom ... - любой текстовый редактор с подсветкой синтаксиса HTML, JS, CSS, PHP

Открыть веб-документ в браузере из Visual Studio - комбинация клавиш Ctrl-Shift-W

Основные HTML-теги
Управление начертанием шрифта
<b>...</b> - полужирное
<i>...</i> - курсивное
<u>...</u> - подчёркнутое

Переход на новую строку
<br/>

Горизонтальная разделительная линия
<hr/>

Заголовки
<h1>...</h1> - Заголовок первого уровня
<h2>...</h2> - Заголовок второго уровня
<h3>...</h3> - Заголовок третьего уровня
<h4>...</h4> - Заголовок четвёртого уровня
...

Абзацы
<p>...</p>

Вставка изображений
<img src="путь к файлу"/>

Гиперссылки
<a href="">Текст гиперссылки</a>

Списки
<ul>..</ul> - маркированный список
<ol>..</ol> - нумерованный список

<li>..</li> - элемент списка

Заголовки, списки, параграфы являются частью семантической разметки страницы

Тег списка используется для представления любых списков на странице. Меню (список гиперссылок) также лучше оформлять в виде HTML-списка.
