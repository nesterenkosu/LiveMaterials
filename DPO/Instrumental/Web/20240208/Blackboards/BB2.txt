CSS - Cascade Stylesheet - таблица стилей (задаёт форматирование для набора HTML-элементов)

правило_отбора {
	параметр1: значение_параметра1;
	параметр2: значение_параметра2;
	...
	параметрN: значение_параметраN;
}

правило отбора задаёт элементы, к которым требуется применить форматирование

p { //Ко всем тегам p
	text-align: justify; // Выравнивание по ширине
	text-indent: 10px; //Отступ красной строки
}

правило отбора (селектор) может быть представлено
- именем тега. В таком случае форматирование будет применено ко всем тегам с таким именем в документе
- идентификатором или классом тега

Выборка по идентификатору (id)
#идентификатор {
}

Выборка по классу (class)
.идентификатор {
}
