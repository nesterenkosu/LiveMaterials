* Вёрстка - компоновка визуальных и логических блоков сайта. Формирование модульной стркутуры интерфейса веб-сайта. Способы вёрстки:
- Табличная
   В качестве модульной структуры используются таблицы (тег table). 
	Достоинства
		Простота и удобство для                   разработчика
         Недостатки
                  Неэффективность с т.з расхода           дискового пространства и интернет-траффика
                  Громоздкий синтаксис табилц
- Слоевая (теги div и span). Контейнерные теги, которые сами по себе не добавляют форматирования тексту, находящемуся внутри них. Но их внешний вид, а также поведение могут быть заданы с помощью таблицы стилей (CSS) и JavaScript.
	div - блочный элемент. Перед текстом внутри себя и после текста внутри себя он добавляет перевод строки. По умолчанию занимает всю доступную ширину окна браузера.
	span - подстановочный элемент. Не добавляет переводов строки, визуально он располагается строго в том месте, где находтся в коде.
Теги div и span по умолчанию не изменяют внешний вид находящегося внутри них текста, но изменяют его поведение относительно текста, находящегося вокруг. Позволяют обозначить некоторый блок текста и применить к нему фоматирование с помощью стилей, либо задать его поведение с помощью javascript.

При слоевой вёрстке для каждого структурного блока создаётся div. Далее для него задаются необходимые параметры с помощью CSS

* Шаблонизация - вынесение повторяющихся составляющих веб-страниц в отдельные файлы.
	- Использование фреймов
	- Использование дирректив на JavaScript
	- Использование дирректив на PHP. Многие CMS предоставляют свои средства для шаблонизации (CMS - Content Management System: Wordpress, Joomla, Drupal, 1c Bitrix, HostCMS)

В CSS каждый тег может иметь идентификатор (id) и класс (class).

- идентификатор должен быть уникальным в рамках всего документа
- класс может применяться к нескольким объектам и задаёт некоторый массив объектов на странице

В CSS отбор по идентификатору - символ #
отбор по классу - символ .

Примеры:
1) выделить красным цветом текст в теге с id = mytest

#mytest {
	color: red;
}

<span id="mytest">Этот текст будет выделен красным</span>

выделить красным цветом текст в элементах с классом mytest

.mytest {
	color: red;
}

Этот текст останется неизменным
<span class="mytest">А этот выделится красным</span>
Этот также будет неизменным
<span>И этот будет неизменным</span>
<span class="mytest">А этот опять будет красным</span>

Задание отступов в CSS
margin - внешний отступ
padding - внутренний отступ

Форматы задания оступов
margin: отступ_сверху отступ_справа отступ_снизу отступ_слева

marign: отступ_сверху_и_снизу отступ_справа_и_слева

margin: отступ_со_всех_сторон

margin-top - отступ сверху
margin-left - отступ слева
