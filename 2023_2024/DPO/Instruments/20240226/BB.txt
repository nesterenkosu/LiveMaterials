Элементы, используемые при вёрстке и их позиционирование.

Контейнеры
- Блочные (block) <div>...</div>
- Подстановочные (inline) <span>...</span>

Контейнер - тег, сам по себе не добавляющий никакого визуального оформления. Он нужен только для структурирования документа и выделения в нём логических элементов. В дальнейшем оформление этих элементов может быть задано с помощью CSS. Также к этим элементам можно получить доступ с помощью JavaScript.

Блочное либо подстановочное поведение может быть также задано для любого HTML-элемента с помощью CSS-свойства display

display: block; элемент ведёт себя как блочный

display: inline; элемент ведёт себя как подстановочный

Блочные
~~~~~~~
Занимают все доступное пространство по ширине

Подстановочные
~~~~~~~~~~~~~~
Их ширина равна ширине содержимого

Свойство float позволяет добавить блочным элементам специальное поведение, используя которое можно применять блочные элементы для вёрстки (формирования столбчатой структуры сайта). Действие свойства float распространяется не только на элемент, где оно задано, но и на следующие за ним элементы. Чтобы отменить действие float в последующих элементах, следует использовать свойство clear: both;

width, height - ширина и высота
margin - внешний отступ
padding - внутренний отступ

margin и padding могут быть заданы в разных форматах
каждый элемент имеет 4 стороны, с каждой из которых может быть отступ
top - верхняя сторона
right - правая сторона
bottom - нижняя сторона
left - левая сторона

Отступы можно указывать явно, например:
margin-left: 3px
margin-top: 2px
padding-right: 7px
padding-bottom: 4px;
(если нужно поменять отступ только с одной или двух сторон)

Можно в сокращённой записи
margin: верх справа внизу слева
padding: верх справа внизу слева
margin: 10px 2px 5px 7px
padding: 10px 2px 5px 7px

margin: отступ_от_вертикальных_сторон отступ_от_горизонтальных_сторон
padding: отступ_от_вертикальных_сторон отступ_от_горизонтальных_сторон
margin: 10px 5px;
padding: 10px 5px;


Настройки позиции элемента

position:

static (по умолчанию)
absolute - позиционирование в координатах относительно левого верхнего угла документа. Координата x задаётся с помощью свойства left. Координата у задаётся с помощью свойства top.
relative -  позиционирование в координатах относительно положения элемента по умолчанию. Координата x задаётся с помощью свойства left. Координата у задаётся с помощью свойства top.

z-index задаёт видимый порядок перекрывающихся слоёв
